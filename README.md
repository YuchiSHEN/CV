# Yuchi Shen — Academic CV

A responsive architecture-style academic CV generated by Jekyll and published with GitHub Pages.

## Edit content

All CV content lives in [`_data/cv`](./_data/cv). Each section has one YAML file. Edit list items in those files, commit, and GitHub Pages will rebuild the website automatically.

See [CONTENT_GUIDE.md](./CONTENT_GUIDE.md) for field definitions and examples.

## Local preview

Install Ruby and Bundler, then run:

```powershell
bundle install
bundle exec jekyll serve
```

Open <http://localhost:4000/WorkSpaceGit/>. A plain `python -m http.server` preview will not process Jekyll data or Liquid templates.
